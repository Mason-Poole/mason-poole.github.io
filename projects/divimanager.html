DiviManager

In September of 2023, I began working as a remote contracted software developer/IT administrator for Shepherds Loft, a local web development and hosting company. My main position was as a developer for DiviManager, a proprietary WordPress plugin, developed in PHP, used to manage other WordPress sites from one interface, removing the need to access each site individually.

DiviManager utilizes a dual-plugin system where a "manager" plugin, hosted on the main site, communicates with several "worker" plugins running on each client's site. Client sites are paired with the manager site using WordPress' built-in integrations system, ensuring the manager site securely connects to the correct client site, and the client site can't be remotely controlled by anything other than the manager site. When the manager site performs an action, it sends a request to the appropriate worker plugin, which parses the request, runs internal functions as needed, and sends a response back to the manager, similar to a RESTful API.

During my time as a developer for DiviManager, I was responsible for developing several key software features, including the following:

  • Remote WordPress core updates
  • Remote site creation
  • Remote plugin and theme deletion
  • Profile-based plugin limits
  • Bulk plugin/theme updates

These features have reduced the time and cost of routine maintenance, expedited the client onboarding process, and provided important information that influenced business decisions. Additionally, I did a significant amount of debugging, using both manual and automated testing to identify issues in the program, which were then logged in Trello and GitHub before being resolved.

My time working at Shepherds Loft has been one of immense growth, giving me a plethora of real world hands-on experience working in a remote company environment. I worked directly with other developers and the business owner to design, document, and implement features that directly impacted the company and its clients. DiviManager is the largest project I've worked on so far, and I was successfully able to come into a project several years in the making, navigate and understand the existing code and structure, and expand upon it while maintaining the company's coding standards. My time at Shepherds Loft has largely shaped me as a professional developer and will forever remain a crucial stage in my career journey.
